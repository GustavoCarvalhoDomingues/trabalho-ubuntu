Guia de Publicações e Servidores

O que é uma Publicação de Páginas Estáticas?

Páginas estáticas são documentos pré-renderizados em HTML, CSS e JavaScript que são enviados diretamente ao navegador do usuário sem processamento no servidor.

  • Exemplos: blogs simples, portfólios pessoais, sites de apresentação.
  • Vantagem: carregamento rápido e menos custos de servidor.
  • Desvantagem: não suportam interatividade avançada sem ajuda de APIs externas.

O que é o Ubuntu?

Ubuntu é uma distribuição Linux de código aberto baseada no Debian. É amplamente utilizada para servidores, desktops e outras aplicações.

Principal Vantagem: O Ubuntu como servidor de sites dinâmicos oferece estabilidade, segurança e eficiência. Ele é gratuito e possui grande suporte da comunidade, sendo ideal para rodar servidores web como Apache e Nginx.

Comandos Básicos do Ubuntu

Abaixo estão alguns comandos básicos que ajudam na administração de servidores Ubuntu:

  • ls - Lista os arquivos e diretórios no diretório atual.
  • cd [diretório] - Navega para o diretório especificado.
  • pwd - Mostra o caminho completo do diretório atual.
  • sudo apt update - Atualiza a lista de pacotes disponíveis.
  • sudo apt upgrade - Instala atualizações para os pacotes.
  • mkdir [nome] - Cria um novo diretório.
  • rm [arquivo] - Remove um arquivo.
  • reboot - Reinicia o servidor.
